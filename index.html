단계별 가이드
설정 방법
01

노션 Integration 만들기

노션 API 접근 권한을 위한 토큰을 발급받아요.

  • notion.so/my-integrations 접속
  • "+ New integration" 클릭 → 이름 입력 (예: monthly-capture)
  • 생성 후 "Internal Integration Secret" 옆 Show 클릭 → 토큰 복사
💡 토큰은 ntn_xxx... 형태예요. 외부에 노출되지 않도록 주의하세요.
02

노션 페이지에 Integration 연결

운영현황 페이지의 상위 페이지에 연결하면 하위 페이지 전체에 자동으로 상속돼요.

  • 노션에서 상위 페이지 열기 (예: 등록환자 DB)
  • 우상단 ··· → Connections → 생성한 Integration 클릭
  • 아카이브 페이지에도 동일하게 연결
⚠️ Integration이 연결되지 않으면 API로 페이지를 읽을 수 없어요. 반드시 두 페이지 모두 연결하세요.
03

페이지 ID 복사

두 페이지의 ID를 각각 복사해 두세요.

# URL에서 마지막 32자리가 페이지 ID
https://www.notion.so/3273a4aad721806e99bdf5ae28c741b3
                       ↑ 이 부분만 복사
04

GitHub 저장소 만들기

코드와 자동화 설정 파일을 저장할 Private 저장소를 만들어요.

  • github.com 로그인 → 우상단 "+" → New repository
  • 이름: notion-monthly-capture
  • Private 선택 후 Create repository 클릭
05

파일 업로드

두 개의 파일을 저장소에 업로드해요.

  • capture.py → 저장소 루트에 업로드
  • capture.yml → .github/workflows/capture.yml 경로로 업로드
💡 폴더 만들기: Add file → Create new file 클릭 후 파일명 입력란에 .github/workflows/capture.yml 을 그대로 입력하면 폴더가 자동 생성돼요.
06

GitHub Secrets 등록

민감한 정보는 코드에 직접 입력하지 않고 Secrets에 안전하게 저장해요.

저장소 → Settings → Secrets and variables → Actions → New repository secret

Secret 이름 값
NOTION_TOKEN Integration 토큰 (ntn_xxx...)
SOURCE_PAGE_ID 운영현황 페이지 ID (32자리)
ARCHIVE_PAGE_ID 아카이브 페이지 ID (32자리)
⚠️ ID 복사 시 앞뒤 공백이나 줄바꿈이 포함되지 않도록 주의하세요. 오류의 주요 원인이에요.
07

수동 테스트 실행

말일을 기다리지 않고 지금 바로 테스트해볼 수 있어요.

  • 저장소 → Actions 탭 클릭
  • 좌측 "노션 운영현황 월말 자동 기록" 클릭
  • 우측 "Run workflow" 버튼 클릭
  • 초록 체크 ✅ 확인 후 노션 아카이브 페이지에서 결과 확인

FAQ
PC가 꺼져 있어도 정말 작동하나요? ↓
네, 완전히 무관해요. GitHub 서버가 자동으로 실행하기 때문에 내 PC 전원, 노션 로그인 여부, 인터넷 연결 상태와 관계없이 작동해요.
데이터베이스가 늘어나도 코드를 수정해야 하나요? ↓
아니요. 코드가 페이지 안의 데이터베이스를 자동으로 모두 찾아서 처리해요. 데이터베이스가 추가되거나 이름이 바뀌어도 별도 수정 없이 자동으로 반영돼요.
Rollup, Relation 값도 저장되나요? ↓
네. Rollup의 합계/평균 등 계산된 최종값이 그대로 저장돼요. 수식(Formula) 결과값도 마찬가지예요. 단, 연결 구조 자체가 아닌 계산된 숫자 결과만 저장돼요.
비용이 드나요? ↓
완전 무료예요. GitHub Actions 무료 플랜은 월 2,000분을 제공하고, 이 작업은 1회 실행에 약 10~20초만 사용해요. 노션 API도 무료 플랜에서 충분히 사용 가능해요.
노션 토큰이 만료되면 어떻게 되나요? ↓
Internal Integration 토큰은 만료되지 않아요. 단, 토큰을 Refresh하거나 Integration을 삭제하면 GitHub Secrets에 새 토큰으로 업데이트해야 해요.
